MonteCarlo.neighbors — Method
neighbors(l::AbstractLattice[, directed=Val(false)])

Returns an iterator over bonds, given as tuples (source index, target index). If directed = Val(true) bonds are assumed to be directed, i.e. both (1, 2) and (2, 1) are included. If directed = Val(false) bonds are assumed to be undirected, i.e. (1, 2) and (2, 1) are assumed to be equivalent and only one of them will be included.
